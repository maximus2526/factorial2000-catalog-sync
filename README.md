# Prom XML Importer for WooCommerce 🚀

**Prom XML Importer** — це легкий, перевірений часом та оптимізований плагін для WordPress, який повністю вирішує проблему синхронізації інтернет-магазинів на WooCommerce із постачальниками, що працюють через Пром. 

Він автоматично імпортує прості та варіативні товари з XML-вивантажень, контролює залишки, актуалізує ціни за розкладом та чистить базу від застарілих позицій.

> 🔥 **Важливо:** Наразі плагін знаходиться у відкритому доступі. Ви можете завантажити та протестувати базовую версію абсолютно безкоштовно. В майбутньому проект планується перевести на **платну основу** з розширеним функціоналом та преміум-підтримкою. Ті, хто підтримає проект на старті, отримають особливі бонуси!

## ⚡ Головні фічі

- **Повний імпорт з Пром:** Коректний парсинг XML-файлів, включаючи складні варіативні товари (розміри, кольори).
- **Розумне оновлення:** Автоматична актуалізація цін та статусів наявності (`instock` / `outofstock`) через WP-CRON у заданий інтервал.
- **Очищення вивантажень:** Фільтрація файлу постачальника для створення нової, "чистої" вигрузки суто з новими позиціями.
- **Сповіщення в Telegram:** Миттєві звіти у ваш чат про успішне завершення імпорту або виникнення помилок на сервері.
- **Оптимізація під Big Data:** Пакетна обробка (batch processing) та регулярне звільнення пам'яті. Легко пережовує каталоги на **10 000+ товарів** навіть на середньому хостингу. (Якщо заходить в timeout запускайте в cron)
- **Екологічність коду:** При деактивації та видаленні плагін повністю вичищає за собою всі налаштування та CRON-задачі, не засмічуючи базу даних `wp_options`.

## 📦 Встановлення

### Варіант 1. Завантаження архівом (Рекомендовано)
1. Завантажте плагін, натиснувши зелену кнопку **Code -> Download ZIP** у верхній частині цієї сторінки.
2. Перейменуйте завантажений архів на `prom-xml-importer.zip`.
3. В адмінці WordPress перейдіть у **Плагіни -> Додати новий -> Завантажити плагін** і виберіть цей архів.
4. Активуйте плагін.

### Варіант 2. Через Git клієнт
```bash
cd wp-content/plugins/
git clone [https://github.com/maximus2526/prom-xml-importer.git](https://github.com/maximus2526/prom-xml-importer.git)
Після цього активуйте його в панелі керування WordPress.
```

## ⚙️ Перше налаштування

1. Після активації в меню адмінки з'явиться новий розділ.
2. Перейти у вкладку **«Робота зі вигрузкою» / «Імпорт»**, вставити URL-посилання на XML від постачальника та запустити первинний імпорт.
3. У вкладці **«Оновлення»** налаштувати інтервал синхронізації цін та залишків, а також вказати Token та ID чату Telegram для сповіщень.
